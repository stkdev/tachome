# RPiタコメーター
Raspberry Piで動かす、タコメーターを作るお遊びプロジェクトです。

**※まだ作成中です**

## 導入
※ Docker化するかも

### 前提
- python3
- SDL ※検証後追記

導入済み

### 仮想環境（任意）

作成
```
python3 -m venv tachome
```

アクティベイト

```
source tachome/bin/activate
```

### 起動準備

ソースコード取得
```
git clone https://github.com/stkdev/tachome.git
```
必要なパッケージ導入
```
pip install -r requirements.txt
```

### 起動
```
python main.py
```
